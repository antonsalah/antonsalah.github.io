Capstone Project

NASA APOD Website

by Anton Salah

Posted on May 5, 2024

The NASA APOD Website is a dynamic web application designed to explore and utilize: front-end design, database caching, real-time data fetching from an external API, and cloud hosting. With the goal of the project to develop through an entire tech stack while also creating a fun and educational application. This website provides users with the ability to view astronomical images along with detailed explanations, based on a user-selected date. Key features and technologies include:

  • Database Caching: implementing SQLite to store data locally, this reduces dependency on API calls and allows for the program to improve as more people use the application.
  • API Integration: Leveraging the free NASA Astrological Photo of the Day (APOD) API to dynamically retrieve the images and descriptions based on the user-selected dates. This integration demonstrates effective real-time data utilization within the application.
  • Cloud Hosting: Utilizing AWS Elastic Beanstalk Service allows the application to leverage auto-scaling capabilities and manages high traffic loads efficiently.


The NASA APOD Website successfully integrates technologies across the software stack to deliver a seamless user experience. The application efficiently checks the local SQLite database for previously requested images and data, reducing API calls and load times. When new data is fetched from the NASA API, it is stored locally, ensuring quicker access on subsequent requests. The interface provides a user-friendly environment where visitors can easily select dates and retrieve the desired astronomical images and descriptions. Additionally, the deployment on AWS Elastic Beanstalk enhances the application's scalability, allowing it to adapt resource allocation dynamically in response to varying traffic loads without the need for manual intervention.

The NASA APOD Website project demonstrates a robust application of web development principles and cloud infrastructure to deliver a functional and educational tool. By leveraging the NASA APOD API, this project not only facilitates access to astronomical data but also encourages public interest and education in astronomy. The project showcased the potential of integrating modern web technologies with public APIs to create informative and interactive experiences. This project can be further expanded by either extending functionality into other NASA Open APIs or by improving the core functionality already established with features such as server scripting, improved UI features, or more robust architectures.

Key Components

The project can be broken down into three essential components:

  • Front-end: The front end utilizes HTML, CSS and Javascript in order to aesthetically display the image and related information
  • Back-end: The webserver was written using Node.js to handle server-side logic, processing the API calls, and querying the database. Along with Express.js to handle page routing.
  • Database: The database uses SQLite to store the date, title, description, and path to the image which is downloaded directly onto disk memory.
